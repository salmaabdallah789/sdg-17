# SDG 17: AI Detective

A browser-based mystery investigation game where players solve social impact mysteries using AI tools.

## How to Play

**Simply open `index.html` in any modern web browser.** No server or installation required!

## Game Features

- **3 Fully Playable Cases:**
  - The Vanishing Classrooms (SDG 4: Quality Education)
  - The Silent Clinics (SDG 3: Good Health & Well-being)
  - The Missing Water (SDG 6: Clean Water and Sanitation)

- **5 AI Tools:**
  - Pattern Finder: Detects anomalies in data
  - Bias Scanner: Identifies missing groups
  - Forecast Lens: Projects trends
  - Text Clue Analyzer: Summarizes statements
  - Map Heat Vision: Shows problem hotspots

- **Interactive Features:**
  - Drag-and-drop deduction board
  - NPC interviews with branching dialogue
  - Evidence collection with card flips
  - Animated score displays
  - Badge system based on performance

## Game Flow

1. **Splash Screen** - Start the investigation
2. **Case Selection** - Choose your mystery
3. **Briefing** - Learn about the case
4. **Investigation Hub** - Explore locations on the map
5. **Evidence Analysis** - Use AI tools to analyze clues
6. **Interrogation** - Interview witnesses
7. **Deduction Board** - Connect evidence to solve the case
8. **Final Accusation** - Present your findings and solution
9. **Results** - See your scores and badge

## Scoring

- **Impact Score**: How well your solution helps people
- **Ethics Score**: Privacy, fairness, and inclusion
- **Feasibility Score**: Realistic resources and partnerships

## Badges

- **Partnership Pro**: High score with strong partnerships (SDG 17 focus)
- **Ethics Guardian**: High ethics score
- **Impact Architect**: High impact score
- **Data Daredevil**: Good overall performance

## Accessibility

- Keyboard navigable
- Reduce motion option available
- Responsive design for mobile devices

## Technical Details

- Pure HTML, CSS, and JavaScript
- No frameworks or dependencies
- Runs entirely in the browser
- Data-driven content (all cases stored as JSON)

Enjoy solving mysteries and learning about AI ethics and social impact!
